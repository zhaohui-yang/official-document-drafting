# 工作方案规范

## 写作规则

- 适用场景：专项行动、阶段性工作、制度落地、项目推进。
- 结构重点：制定背景、总体要求、工作目标、主要任务、实施步骤、保障措施。
- 写作提醒：任务要拆清、步骤要可执行、保障措施要能落地。
- 分段职责：`制定背景` 写为什么要制定方案、依据和现实需求；`总体要求` 写指导思想、基本原则和总体安排；`工作目标` 写预期成效、量化指标或时间要求；`主要任务` 写具体工作、责任分工和完成标准；`实施步骤` 写阶段安排和时间节点；`保障措施` 写组织领导、资源保障、督导考核和纪律要求。
- 事实要求：不得编造未提供或未核实的事实、数据、时间、地点、机构名称、人员身份、文件号和政策依据；信息不足时标注“待核实”或保留占位符。
- 常用结尾：可不单独设置套话式结尾，通常以措施或附件收束。

## 版式要求

- 基线要求：遵循共享总规则中的“版式与导出”。
- 标题通常居中排布，可用 2 号小标宋体，常见格式为“[事项]工作方案”或“[事项]实施方案”。
- 一般不设主送单位；如作为通知附件或专项文件，可在标题下补充适用范围或执行期限说明。
- 正文通常用 3 号仿宋体，首行缩进 2 字符；“总体要求、目标任务、实施步骤、保障措施”宜分层写清。
- 工作方案常含附件、流程表、责任分工等内容，正文中的时间节点和责任主体应单独成句或成条。
- 若方案要直接用于发文或会议印发，最终版式仍应与本单位成套模板保持一致。

[填写具体工作内容、责任分工和完成标准。]

二、[任务二]

[填写具体工作内容、责任分工和完成标准。]

## 实施步骤
