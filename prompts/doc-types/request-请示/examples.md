# 请示示例

## 适用任务
- 用户说：请就开展“我的刀盾”传播案例梳理工作，向上级提出经费和组织支持申请。

## 推荐题名
- 关于申请开展“我的刀盾”传播案例梳理工作的请示

## 最小骨架
```markdown
# 关于申请开展“我的刀盾”传播案例梳理工作的请示

## 主送单位

[主送单位]：

　　一、请示背景
　　二、请示事项
　　三、相关说明

妥否，请批示。

## 落款

[发文单位]
[日期]

## 附注

（联系人：[联系人] 联系电话：[固定电话]，[手机号]）
```

## 常见误写
- 一文一事，不要在一份请示中并列提出多个不相关事项。
- 不要把请示写成汇报性报告。
