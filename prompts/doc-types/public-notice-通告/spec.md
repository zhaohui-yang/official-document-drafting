# 通告规范

## 写作规则

- 适用场景：在一定范围内公布应当遵守或者周知的事项。
- 结构重点：发布缘由、通告事项、执行要求、落款。
- 写作提醒：强调“周知”和“遵守”，适用范围通常比公告更具体。
- 事实要求：不得编造未提供或未核实的事实、数据、时间、地点、机构名称、人员身份、文件号和政策依据；信息不足时标注“待核实”或保留占位符。
- 常用结尾：特此通告。

## 版式要求

- 基线要求：遵循共享总规则中的“版式与导出”。
- 标题通常用 2 号小标宋体居中排布，宜写明通告事项和适用范围。
- 通告一般不设主送单位，正文可按事项、区域、时限、要求分条列明。
- 正文用 3 号仿宋体，首行缩进 2 字符；面对社会公众时，条目表达要比内部文件更直接清楚。
- 处罚依据、执行时间、咨询电话、责任单位等信息宜单独成条，不要埋在叙述段中。
- 发布单位和日期通常置于文末右侧；公告性文本最终仍应使用本单位正式发布模板。

## 模板

```markdown
# 通告模板

## 标题

关于[事项名称]的通告

## 发布缘由

[说明通告发布的背景、原因或目的。]

## 通告事项

一、[事项一]  
[写明周知或应当遵守的具体事项。]

二、[事项二]  
[补充要求。]

## 执行要求

[写明适用范围、执行时间或责任要求。]

## 落款

[发文单位]  
[日期]
```
