# 工作总结规范

## 写作规则

- 适用场景：阶段性复盘、年度总结、专项工作总结。
- 结构重点：基本情况、主要做法、工作成效、存在问题、下一步打算。
- 写作提醒：成效和问题都要写，避免只报喜不报忧。
- 事实要求：不得编造未提供或未核实的事实、数据、时间、地点、机构名称、人员身份、文件号和政策依据；信息不足时标注“待核实”或保留占位符。
- 常用结尾：下一步将持续抓好贯彻落实。

## 版式要求

- 基线要求：遵循共享总规则中的“版式与导出”。
- 标题通常居中排布，可用 2 号小标宋体，常见格式为“[年度/阶段][事项]工作总结”。
- 一般不设主送单位；如作为正式报送件，可在标题下说明报送对象或报送日期。
- 正文通常用 3 号仿宋体，首行缩进 2 字符；“主要做法、取得成效、存在问题、下一步打算”宜分层表达。
- 数据、案例、问题和经验做法应分别成段，避免通篇套话；一级标题可用黑体突出各板块。
- 如要转为述职材料、汇报材料或考核材料，建议在总结初稿基础上再按目标场景细化版式。
