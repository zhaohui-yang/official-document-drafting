# 纪要示例

## 适用任务
- 用户说：请根据会议材料整理一份关于研究“我的刀盾”网络传播情况的专题会议纪要。

## 推荐题名
- 关于研究“我的刀盾”网络传播情况的专题会议纪要

## 最小骨架
```markdown
# 关于研究“我的刀盾”网络传播情况的专题会议纪要

## 会议基本信息

时间：[日期]
地点：[地点]
主持人：[主持人]

## 出席

[出席人员]

## 参加

[参加人员]

## 记录

[记录人]

　　会议认为，……
　　会议决定，一、……二、……三、……

## 版记

主送：[主送单位]
抄送：[抄送单位]
审核：[审核人]
```

## 常见误写
- 不要把“会议认为”“会议决定”单独写成又一层标题。
- 不要虚构会议表态、参会人员或议定结论。
