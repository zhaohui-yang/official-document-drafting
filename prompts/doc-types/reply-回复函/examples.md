# 回复函示例

## 适用任务
- 用户说：请就对方来函咨询“我的刀盾”传播素材共享事宜，起草一份回复函。

## 推荐题名
- 关于“我的刀盾”传播素材共享事宜的回复函

## 最小骨架
```markdown
# 关于“我的刀盾”传播素材共享事宜的回复函

## 主送单位

[来函单位]：

　　你单位《关于……的来函》收悉。现回复如下：
　　一、……
　　二、……

此复。

## 落款

[发文单位]
[日期]

## 附注

（联系人：[联系人] 联系电话：[固定电话]，[手机号]）
```

## 常见误写
- 不要漏写“收悉”或对应回复对象。
- 不要把回复函写成新的请示或通知。
