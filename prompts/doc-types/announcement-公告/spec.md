# 公告规范

## 写作规则

- 适用场景：向国内外宣布重要事项或者法定事项。
- 结构重点：公告事项、生效或执行说明、落款。
- 写作提醒：公开性和严肃性强，仅在事项足够重大时使用。
- 分段职责：`公告事项` 重点写向社会公开宣布什么、适用对象是谁、核心内容是什么；`执行说明` 重点写何时生效、如何执行、如何办理、违反后如何处理或向何处咨询，不写内部酝酿过程。
- 事实要求：不得编造未提供或未核实的事实、数据、时间、地点、机构名称、人员身份、文件号和政策依据；信息不足时标注“待核实”或保留占位符。
- 常用结尾：特此公告。

## 版式要求

- 基线要求：遵循共享总规则中的“版式与导出”。
- 标题通常用 2 号小标宋体居中排布，可直接写“公告”或“关于[事项]的公告”。
- 公告一般不设主送单位，正文可按事项分条列出，正文用 3 号仿宋体，首行缩进 2 字符。
- 一级标题通常用黑体，二级标题通常用楷体；如正文较短，可不强行分层。
- 发文单位和日期通常置于文末右侧；对外公告如已有固定署名格式，以发布主体模板为准。
- 导出 Word 时优先保持标题居中、条款分明，不要通过手工空格模拟版心或居中。

## 模板

```markdown
# 公告模板

## 标题

关于[事项名称]的公告

## 公告事项

[面向社会或特定范围公开宣布的重要事项或法定事项。]

## 执行说明

[写明生效时间、执行要求、相关说明或注意事项。]

## 落款

[发文单位]  
[日期]
```
