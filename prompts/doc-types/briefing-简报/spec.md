# 简报规范

## 写作规则

- 适用场景：简要报送动态、会议情况、阶段成果、当前新闻整理。
- 结构重点：标题、导语、主要信息、工作建议或提示。
- 要素分层：`必备` 为标题、导语、主要信息；`常见` 为工作建议或提示；`条件项` 为报送范围、风险提示、附件；`地方或系统样式` 为期数、编发单位、编发日期等版头信息，是否保留以用户模板为准。
- 写作提醒：优先区分“事实摘要”和“提示建议”，篇幅宜短、信息密度要高。
- 分段职责：`导语` 用一段话交代时间、地点、主体和最重要事实；`主要信息` 按条写最新进展、关键数据、典型情况或重要动向；`工作建议或提示` 写需要持续关注的事项、风险提醒或后续提示，不能用大段空话代替信息。
- 常见组织方式：内部简报宜坚持“一期一主题”“一报一重点”，围绕一个核心事项集中编发，不把多个无关事项硬拼成同一篇简报。
- 地方或系统样式：部委和省级系统样例中的简报，常带 `第[期]期`、编发单位、编发日期以及 `报：/送：/发：` 等尾部信息；是否保留应以用户模板或系统习惯为准，不机械套用到所有简报。
- 简报较适合通过 `附件`、`附图` 或配套图表承载照片、截图、传播热度图、现场图和流程图；图件宜有简短说明，不把图片当作无说明配图。
- 事实要求：不得编造未提供或未核实的事实、数据、时间、地点、机构名称、人员身份、文件号和政策依据；信息不足时标注“待核实”或保留占位符。
- 常用结尾：可不固定套话，以提示事项或报送范围收束。

## 版式要求

- 基线要求：遵循共享总规则中的“版式与导出”。
- 标题通常居中排布，可用 2 号小标宋体或与本单位简报模板一致的标题字体。
- 简报可在标题下增设期数、编发单位、日期等信息行；如无现成模板，可保持简洁单列。
- 正文通常用 3 号仿宋体，首行缩进 2 字符；新闻动态型简报宜短段落、快节奏、少空话。
- 小标题建议用黑体或楷体，信息点较多时可采用“一、（一）”或项目符号式分段。
- 若形成内部刊发件，最终版头、期号、编发栏等仍以本单位既有简报版式为准。

## 模板

```markdown
# 简报模板

## 标题

[事项名称]简报

## 一、导语

[用一段话概括时间、地点、主体和主要情况。]

## 二、主要信息

（一）[信息点一]

[填写事实摘要。]

（二）[信息点二]

[填写事实摘要。]

## 三、工作建议或提示

[填写后续关注事项、工作提示或简要建议。]

## 报送范围（可选）

[填写报送对象或阅知范围。]

## 附件（可选）

附件1：[图片或图表标题]

图1：[图片标题]
[图片描述]
说明：[说明文字]
来源：[来源]
```
