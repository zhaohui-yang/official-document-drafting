# offline / raw-materials

本目录用于存放离线场景下已经下载、整理或人工汇编的原始材料。

使用建议：

- 原始材料尽量保持“长而全”，不要过早删减。
- 每份材料都应标注来源、日期和基本说明，便于后续复核。
- 离线写作时，先从本目录挑选或汇总原始材料，再为具体文种提炼 `materials.md`。
- 如素材较长，可同时在各文种目录中保留一份“提炼摘要”，不要直接把整份原始材料塞进最终成稿。
